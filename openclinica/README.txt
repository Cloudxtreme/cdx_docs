Dynamic CRF's:
OpenClinica Amethyst, the new Milestone and next feature release of
OpenClinica, will support dynamic logic within a single CRF.  For
example, if the data entry person selects Female as the sex, then the
questions about pregnancy are shown on the screen.  If the user selects
Male, they will never see the pregnancy questions.
